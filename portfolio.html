Pandora's Box

A Broken Computer is Never a Dead Computer

When it comes to computers, there are endless possibilities how to use them. For this project, I took a broken laptop and turned it into my own multi-purpose server. It currently hosts a fully responsive web page which acts as a portal to my personal media server and cloud storage - both are simultaneously hosted directly on the server.
